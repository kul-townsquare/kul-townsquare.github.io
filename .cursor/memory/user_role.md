# 用户角色

用户是《染：钟楼谜团》（Blood on the Clocktower）的**玩家**，维护这个 fork 是为了自己/朋友打本子用。

## 与作者的关系

- **零沟通渠道**：剧本 JSON 都是从作者处拿到的成品
- **不能要求作者改**：任何"请作者用标准 ID / 标准 schema"的方案都不现实
- **用户是下游消费者**，不是工具链上游

## 协作偏好

- 以**软件工程视角**思考问题，不要滥用科研模板
- 明确区分：什么是用户契约（不能破坏）、什么是内部实现（可以自由重构）
- 重大改动（Level 3）手动验证后再推；生产分支（`main`）绝不由 AI 推
- 提交信息**不要**带 co-author 或 Claude 署名
- 规划 / 规则 / 讨论用中文，代码注释和 commit 用英文
